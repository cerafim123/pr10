Правила Финала этапа


Финалисты, подавшие заявки на участие в турнире, будут разбиты на команды по 3 человека. Команде будет предложено решить 3 набора из 3 японских кроссвордов, одинаковых для каждой команды.
Из трех заданий, которые получит каждый участник, первое будет черно-белым кроссвордом, второе – цветным и третье – двуцветным.
Победит команда, участники которой (суммарно) потратят наименьшее время на решение задания:
  • Участники команды, занявшей первое место, получат по 400 рублей.

  • Участники команды, занявшей второе место, получат по 300 рублей.

  • Участники команды, занявшей третье место, получат по 200 рублей.

Итоги соревнования в личном зачете подводятся в конце года.
